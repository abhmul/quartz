Counterexample showing that optimizing BLER is not the same as optimization CE

Last updated April 25, 2022

    Table of Contents

    Dates: 2022-03-10

    Consider the following setup:

    1. Probability Space $(\Omega, \mathcal{F}, \mathbb{P})$
    2. Random Variable $U: \Omega \to {1,2}$. with uniform probability. That is $\mathbb{P}(U=1) = \mathbb{P}(U=2) = \frac{1}{2}$.
    3. Measureable Functions
      1. Encoder $f: {1, 2} \to [n]$
      2. Noise Function $q: [n] \times \Omega \to [n]$
      3. Likelihood Function $l: [n] \to [0,1]^{2}$. That is $l(i)_{j} = \mathbb{P}(U=j|Y=i)$.
      4. Generic soft decoder $g: [n] \to [0,1]^{2}$. $g$ returns a Probability Mass Function on $[2]$.
      5. Hard thresholder $h: [0,1] \to [2]$, where $h(x) = \begin{cases} 1 &\text{if }x \geq 0.5\ 0 &\text{otherwise} \end{cases}$
    4. Denote Random Variables
      1. $X = f \circ U$. This is the encoded value.
      2. $Y(\omega) = q(X(\omega), \omega)$. This is the noisy value
      3. $L = l \circ Y$. This a random variable of the likelihood for $U=1$ conditioned on $Y$.
      4. $R = g \circ Y$. This is a random variable of the soft decoded value conditioned on $Y$.
      5. $\hat{U} = h \circ L$. This is a random variable representing ML decoded value.
      6. $\hat{V} = h \circ R$. This is a random variable representing the decoded value determined by hard thresholding of the soft decoded value.

    Definition 1: The Block Error Rate of an encoder $f$, soft decoder $g$ given some noise model $q$ is $$BLER(f, g|q) = \mathbb{E}\big[ 1_{\hat{V} \neq U} \big]$$

    Note that this is really just the Risk for the Classification Problem of classifying $U$ using $Y$. So it follows that the Bayes Classifier minimizes BLER.

    Definition 2: The Cross-Entropy of an encoder $f$, soft decoder $g$ given some noise model $q$ is $$CE(f, g|q) = \mathbb{E}\Big[-\sum\limits_{j \in [2]} 1_{U=j} \log g(Y)_{j} \Big]$$

    Recall that we can write the Cross-Entropy as $$CE(f, g|q) = \mathbb{H}(U|Y) + \mathbb{E}\big[ D_{KL}(\mathbb{P}(U|Y) || g(Y) ) \big]$$

    From here we see that $g=l$ minimizes $CE(f, g|q)$ for any choice of $f, q$. $l$ induces the Bayes Classifier using the hard thresholder $h$, so we can modify our definitions to focus on $f$:

    Definition 1.1: The Block Error Rate of an encoder $f$ given some noise model $q$ is $$BLER(f|q) = \mathbb{E}\big[ 1_{\hat{U} \neq U} \big] = \mathbb{P}(\hat{U} \neq U)$$

    and

    Definition 2.1: The Cross-Entropy of an encoder $f$ given some noise model $q$ is $$CE(f|q) = \mathbb{E}\Big[-\sum\limits_{j \in [2]} 1_{U=j} \log l(Y)_{j} \Big ]= \mathbb{H}(U|Y)$$

    We can simplify our definition of Block Error Rate by observing that $$BLER(f|q) = \mathbb{P}(\hat{U} \neq U) = 1 - \frac{\mathbb{P}(\hat{U} =1| U=1)}{2} - \frac{\mathbb{P}(\hat{U} =0| U=0)}{2}$$

    We can decompose these probabilities as

    $$\begin{align*} \mathbb{P}(\hat{U}=1|U=1) &= \mathbb{P}(1_{\mathbb{P}(U=1|Y) \geq \mathbb{P}(U=2|Y)}=1|U=1)\\&=\sum\limits_{y \in [n]} \mathbb{P}(1_{\mathbb{P}(U=1|Y) \geq \mathbb{P}(U=2|Y)}=1|Y=y)\mathbb{P}(Y=y|U=1)\\&=\sum\limits_{y \in {i:\mathbb{P}(U=1|Y=i) \geq \mathbb{P}(U=2|Y=i)}} \mathbb{P}(Y=y|U=1)\\&=\sum\limits_{y \in {i:\mathbb{P}(U=1|Y=i) \geq \mathbb{P}(U=2|Y=i)}} \mathbb{P}(Y=y|X=f(1)) \end{align*}$$

    Let’s rewrite $\mathbb{P}(U=j|Y=y)$ using Bayes Theorem $$\begin{align*} \mathbb{P}(U=j|Y=y) &= \frac{\mathbb{P}(Y=y|U=j)\mathbb{P}(U=j)}{\sum\limits_{k \in [2]} \mathbb{P}(Y=y|U=k)\mathbb{P}(U=k)}\\&= \frac{\frac{\mathbb{P}(Y=i|X=f(j))}{2}}{\frac{1}{2}\sum\limits_{k \in [2]} \mathbb{P}(Y=y|X=f(k))}\\&= \frac{\mathbb{P}(Y=i|X=f(j))}{\sum\limits_{k \in [2]} \mathbb{P}(Y=y|X=f(k))} \end{align*}$$

    Thus, $$\begin{align*} &\mathbb{P}(U=1|Y=i) \geq \mathbb{P}(U=2|Y=i)\\&\Rightarrow \frac{\mathbb{P}(Y=i|X=f(1))}{\sum\limits_{k \in [2]} \mathbb{P}(Y=y|X=f(k))} \geq \frac{\mathbb{P}(Y=i|X=f(2))}{\sum\limits_{k \in [2]} \mathbb{P}(Y=y|X=f(k))}\\&\Rightarrow \mathbb{P}(Y=i|X=f(1)) \geq \mathbb{P}(Y=i|X=f(2)) \end{align*}$$ So $$\begin{align*} \mathbb{P}(\hat{U}=1|U=1) &=\sum\limits_{y \in {i:\mathbb{P}(Y=i|X=f(1)) \geq \mathbb{P}(Y=i|X=f(2))}} \mathbb{P}(Y=y|X=f(1)) \end{align*}$$ And likewise for $U=2$. Thus

    $$2BLER(f|q) = 2 - \sum\limits_{y \in {i:\mathbb{P}(Y=i|X=f(1)) \geq \mathbb{P}(Y=i|X=f(2))}} \mathbb{P}(Y=y|X=f(1)) - \sum\limits_{y \in {i:\mathbb{P}(Y=i|X=f(1)) < \mathbb{P}(Y=i|X=f(2))}} \mathbb{P}(Y=y|X=f(2)) \checkmark$$

    Now, to simplify our Conditional Entropy, we can use our Bayes Theorem computation from above $$\begin{align*} CE(f|q) &= \mathbb{H}(U|Y)\\&=-\sum\limits_{y \in [n]} \mathbb{P}(Y=y) \sum\limits_{j \in [2]} \frac{\mathbb{P}(Y=y|X=f(j))}{2\mathbb{P}(Y=y)} \log \frac{\mathbb{P}(Y=y|X=f(j))}{2\mathbb{P}(Y=y)}\\&=-\frac{1}{2} \sum\limits_{y \in [n]} \sum\limits_{j \in [2]} \mathbb{P}(Y=y|X=f(j)) \log \frac{\mathbb{P}(Y=y|X=f(j))}{2\mathbb{P}(Y=y)}\\&=-\frac{1}{2} \sum\limits_{y \in [n]} \sum\limits_{j \in [2]} \mathbb{P}(Y=y|X=f(j)) \log \frac{\mathbb{P}(Y=y|X=f(j))}{\sum\limits_{k \in [2]} \mathbb{P}(Y=y|X=f(k))} \checkmark\\\end{align*}$$

    We want to construct $q$ so the 2 best possibilities are

    $$\mathbb{P}(\hat{U}=1|U=1) = .5, \mathbb{P}(\hat{U}=2|U=2) = .7$$ and $$\mathbb{P}(\hat{U}=1|U=1) = .505, \mathbb{P}(\hat{U}=2|U=2) = .694$$

    Note: This choice above does not use the right reasoning, since cross-entropy is not in terms of $\mathbb{P}(\hat{U}=j|U=j)$. However, I got lucky and my constructed noise model ended up working out.

    Our noise model can be represented as a Markov Model on $n$ vertices. This can be represented as $n \times n$ transition matrix (so each row must sum to 1). $f$ will select two vertices (corresponding to rows). We can denote this matrix $T = (p_{ij})$.

    Let’s try this with $n=4$. Then

    $$T = \begin{pmatrix}p_{11}&p_{12}&p_{13}&p_{14} \\p_{21}&p_{22}&p_{23}&p_{24} \\p_{31}&p_{32}&p_{33}&p_{34} \\p_{41}&p_{42}&p_{43}&p_{44}\end{pmatrix}$$

    Then our expressions for $BLER$ and $CE$ are

    $$\begin{align*} 2BLER(f|q) &= 2 - \sum\limits_{y \in {i:\mathbb{P}(Y=i|X=f(1)) \geq \mathbb{P}(Y=i|X=f(2))}} \mathbb{P}(Y=y|X=f(1)) - \sum\limits_{y \in {i:\mathbb{P}(Y=i|X=f(1)) < \mathbb{P}(Y=i|X=f(2))}} \mathbb{P}(Y=y|X=f(2))\\&=2 - \sum\limits_{y \in {i:p_{f(1)i} \geq p_{f(2)i}}} p_{f(1)y} - \sum\limits_{y \in {i:p_{f(1)i} < p_{f(2)i}}} p_{f(2)y}\\&=2 - \sum\limits_{y \in [4]} \max(p_{f(1)y},p_{f(2)y}) \end{align*}$$

    and $$\begin{align*} 2CE(f|q) &= -\sum\limits_{y \in [n]} \sum\limits_{j \in [2]} \mathbb{P}(Y=y|X=f(j)) \log \frac{\mathbb{P}(Y=y|X=f(j))}{\sum\limits_{k \in [2]} \mathbb{P}(Y=y|X=f(k))}\\&= -\sum\limits_{y \in [n]} \sum\limits_{j \in [2]} p_{f(j)y} \log \frac{p_{f(j)y}}{\sum\limits_{k \in [2]} p_{f(k)y}} \end{align*}$$

    We’ll try the matrix

    $$T = \begin{pmatrix}.24&.26&.2605&.2395 \\.15&.15&.35&.35 \.24&.26&.2605&.2395 \\.056&.343&.25&.351 \\end{pmatrix}$$
