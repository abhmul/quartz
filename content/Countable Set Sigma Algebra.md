# Statement
Let $S$ be a [[Set]]. Then the [[Set|Collection]]
$$\mathcal{M} = \{E \subset S : E \text{ is countable or }E^{C} \text{ is countable}\}$$
is a [[Sigma Algebra]]. We call it the [[Countable Set Sigma Algebra]].
