# Definition
Suppose $F$ is a [[Distribution Function]]. Then we call it a **Probability Distribution Function** if
$$\begin{align*}
&F(\infty) = 1\\
&F(-\infty) = 0\\
\end{align*}$$
We may also call $F$ a
- [[Proper Distribution Function]]
- [[Non-defective Distribution Function]]
