# Definition
The [[Kroenecker Delta Function]], $\delta: \mathbb{N} \times \mathbb{N} \to \{0, 1\}$ is defined as
$$\delta_{ij} := \delta(i, j) := \begin{cases} 1  & \text{if } i = j \\
0  & \text{otherwise} \end{cases}$$
