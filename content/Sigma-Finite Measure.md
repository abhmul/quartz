# Definition
Let $(X, \mathcal{M}, \mu)$ be a [[Measure Space]]. We call $\mu$ a [[Sigma-Finite Measure]] if there exists a [[Countable]] [[Partition]] of $X$, $\{P_{n}\}_{n \in \mathbb{N}}$, so that $\mu(P_{n}) < \infty$ for all $n \in \mathbb{N}$.
