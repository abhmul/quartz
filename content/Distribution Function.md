# Definition
Let $F: \mathbb{R} \to \mathbb{R}$. Then $F$ is a distribution function if
1. $F$ is [[Non-Decreasing Function]]
2. $F$ is [[Right-Continuous]]

Denote $F(\infty)= \lim_{x \to \infty}F(x)$ and $F(-\infty)= \lim_{x \to -\infty}F(x)$.
