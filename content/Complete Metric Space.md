# Definition
Let $(M, d)$ be a [[Metric Space]]. We say $M$ is a [[Complete Metric Space]] if for all [[Cauchy Sequence]]s $({x}_{n})_{n=1}^{\infty} \subset M$, there exists $x \in M$ so that $x_{n} \to x$.
