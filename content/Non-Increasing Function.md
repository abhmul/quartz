# Definition
Let $f: X \to Y$, where $X$ and $Y$ are [[Total Ordering|totally ordered]]. Then $f$ is a [[Non-Increasing Function]] if $\forall x, y \in X$:

$$x \leq y \Rightarrow f(x) \geq f(y)$$
Note that if $x > y$, we know that $y \leq x$ so $f(x) \leq f(y)$.
